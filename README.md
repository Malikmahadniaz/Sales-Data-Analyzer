# 📊 Sales Data Analyzer (Pandas + NumPy)

A clean and practical data analysis project built using **Pandas** and **NumPy**. This project focuses on exploring, cleaning, and analyzing an e-commerce sales dataset to extract meaningful business insights.

---

## 🚀 Project Overview

The **Sales Data Analyzer** is designed to strengthen core data analysis skills using only essential Python libraries. It walks through the complete workflow of a real-world dataset:

* Data loading
* Data exploration
* Data cleaning
* Feature handling
* Statistical analysis
* Outlier detection
* Insight generation

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**
* **NumPy**

---

## 📂 Dataset

* File: `global_ecommerce_sales.csv`
* Contains sales-related data such as:

  * Order details
  * Product information
  * Pricing & quantity
  * Dates

---

## ⚙️ Features & Tasks Covered

### 🔹 Data Exploration

* Understanding dataset structure (`shape`, `columns`, `info`)
* Summary statistics (`describe`)
* Missing value analysis

### 🔹 Data Cleaning

* Handling null values
* Removing duplicates
* Converting data types (e.g., dates)

### 🔹 Data Processing

* Working with numerical and categorical data
* Creating derived columns
* Filtering and grouping data

### 🔹 Analysis

* Sales trends
* Revenue calculations
* Aggregations using `groupby`

### 🔹 Advanced Tasks

* Outlier detection
* Statistical insights using NumPy
* Data consistency checks

## 📈 Sample Insights You Can Generate

* Top-performing products

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Malikmahadniaz/sales-data-analyzer.git
   ```

2. Navigate to the project folder:

   ```bash
   cd sales-data-analyzer
   ```

3. Install dependencies (if needed):

   ```bash
   pip install pandas numpy
   ```

4. Run the notebook:

   ```bash
   jupyter notebook Sales_Data_Analyzer.ipynb
   ```
